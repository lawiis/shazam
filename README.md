# ⚡ SHAZAM TOOLS v4

Terminal-based multi utility toolkit built with Python.  
All-in-one tool for system info, networking checks, hashing, and basic encryption.

---

## 🚀 Features

- 🖥️ **System Information**
- 🌐 **Public IP Checker**
- 🔎 **Multi Port Scanner**
- 🔐 **Hash Generator** (MD5, SHA1, SHA256)
- 🔒 **Base64 Encoder / Decoder**
- 🧠 **XOR Encryption & Decryption**
- 🎨 **Colored Terminal Interface**
- 🧹 **Auto Clear Terminal**
- 👋 **Clean Exit on Ctrl + C**
