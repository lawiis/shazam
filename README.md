## 📋 Requirements

- **Python 3.8+**
- **pip** (Python package manager)
- **OS** : Linux/macOS/Windows (with terminal)

---

## 🚀 Installation

### 1️⃣ Clone Repository
```bash
git clone https://github.com/lawiis/shazam.git
cd shazam
```

### 2️⃣ Install Dependencies
```bash
pip install colorama requests
```

### 3️⃣ Run The Tool
```bash
python3 shazam.py
```

> 💡 **Note:** On Linux/macOS, you might need to use `sudo` for certain network operations:
> ```bash
> sudo python3 shazam.py
> ```

---

## 🎮 Usage Guide

1. **Launch** the tool using the command above
2. **Choose** from the interactive menu (0-7)
3. **Follow** the on-screen instructions
4. **Press Ctrl+C** anytime to exit gracefully

### Menu Options:
- `[1]` System Information - View your system details
- `[2]` Public IP Checker - Get your public IP and location
- `[3]` Multi Port Scanner - Scan open ports on a target
- `[4]` Hash Generator - Generate MD5, SHA1, SHA256 hashes
- `[5]` Base64 Encoder/Decoder - Encode/decode Base64 strings
- `[6]` XOR Encryption - Encrypt/decrypt text with XOR cipher
- `[7]` Clear Screen - Clear the terminal
- `[0]` Exit - Exit the program

## 🗑️ Uninstall

Want to remove Shazam Tools? Follow these steps:

### Option A: Manual Removal
```bash
# Remove the cloned directory
rm -rf shazam

# Optional: Uninstall Python packages (if no longer needed)
pip uninstall colorama requests -y
```

### Option B: Complete Cleanup
```bash
# Remove directory and all related files
cd ..
sudo rm -rf shazam

# Check and remove Python packages
pip uninstall colorama requests -y

# Clean pip cache (optional)
pip cache purge
```
